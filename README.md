# Tablet-OS
Takani BSD Tablet Device OS

# Licensing
Tablet OS is a derivative of Takani BSD -- Aimed at general purpose use in portable electronics. It contains components derived from the FreeBSD operating system and is provided at no cost.

Tablet OS is for public use and carries no warranty.

# Release Engineering
Tablet OS is released as "Periodic Increments"; Based on tidings and forecasts of low bug reports.

# Statement of No Competition
At no contention to the originating community; FreeBSD is being forked and committed to a new development future. All originating source code will be replaced and rewritten. Tablet OS is not a competitive or commercial venture.

# Providership Statement
Tablet OS is established and arranged through the TKAS Foundation. Participation by the community is voluntary and welcomed.
